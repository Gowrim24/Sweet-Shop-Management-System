# Sweet Shop Management System

## Project Description
This project is a backend API for managing a Sweet Shop. It allows users to register, login, manage sweets, and perform CRUD operations securely using REST APIs.

## Technologies Used
- Python
- Flask
- SQLite
- JWT Authentication
- Pytest
- Swagger API Documentation

## Features
- User Registration and Login
- JWT-based Authentication
- Add, View, Update, Delete Sweets
- Secure API access
- API documentation using Swagger
- Automated testing using Pytest

## API Endpoints

### Authentication
- POST /auth/register
- POST /auth/login

### Items (Sweets)
- GET /items/
- POST /items/
- PUT /items/<id>
- DELETE /items/<id>
